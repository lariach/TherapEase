

### Installation

1. Clone the repository
   ```sh
   git clone https://github.com/lariach/TherapEase
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Build Project
   ```sh
   npm run build
   ```
4. Run Project
   ```sh
   npm run start-dev
   ```
5. Run Local Server
   ```sh
   npm run start-server
   ```
